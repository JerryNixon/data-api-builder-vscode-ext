# MCP Inspector Reference

## Purpose

Use MCP Inspector to validate MCP connectivity and tool behavior against DAB.

## Local guidance

- Ensure DAB `/mcp` endpoint is enabled and healthy.
- Use the Inspector URL format expected by your runtime.
- For containerized local debugging, ensure service-name routing (not `localhost`) between containers.
