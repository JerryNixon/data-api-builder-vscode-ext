# Agent Deployment Scripts

## Purpose

This folder contains script templates used by the DAB developer agent for deployment-oriented guidance.

## Scope

- Azure Container Apps deployment templates
- Production-hardened variant templates
- Dockerfile sample for custom DAB image build

## Usage model

- Treat scripts as templates.
- Require explicit variable review before execution.
- Keep script behavior consistent with `deployment-scripts.md`.
