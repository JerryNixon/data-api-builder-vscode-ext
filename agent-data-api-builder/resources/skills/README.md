# DAB Skills Catalog

This folder contains reusable skill packs for common Data API Builder workflows.

## Design contract

- Skills are the broader knowledge layer.
- The `DAB Developer` agent is the focused execution layer.
- Guidance must be generic for typical developers (not demo-persona specific).

## Skill groups

- Core: CLI, config, auth, MCP
- Local runtime: Docker, Aspire (+ SQL Commander / SQL Projects / MCP Inspector)
- Cloud runtime: Azure deployment (+ Azure SQL Commander / Azure MCP Inspector)
- Meta: demo-ops conventions, creating agent skills
